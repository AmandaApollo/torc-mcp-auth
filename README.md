# Workshop: Secure AI with Apollo MCP: Build Trusted Agentic Systems

Workshop for Torc Community given Nov 2025. 

## Agenda

This workshop covers adding auth to MCP Servers and role based access using schema directives.  Attendees left with login for their MCP tools, admin and guest scoped data access, and safe execution configuration for mutations. If you are interested in learning more or would like to chat about a workshop like this for your community, please contact Amanda.
